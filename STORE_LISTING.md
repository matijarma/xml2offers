# Chrome Web Store Copy - Izrada ponuda

## Extension Name

Izrada ponuda

## Short Description (EN, <=132 chars)

Create professional offer PDFs locally with issuer profiles, VAT totals, instant preview, and download.

## Short Description (HR, <=132 chars)

Izradjujte profesionalne PDF ponude lokalno: izdavatelji, stavke, PDV izracun, pregled i preuzimanje.

## Detailed Description (EN)

Create polished business offers directly in your browser and export them to PDF in seconds.

`Izrada ponuda` helps you prepare offers faster with a practical workflow:

- Manage multiple issuer profiles
- Import issuer details from e-invoice XML when needed
- Build offers with line items, quantities, unit prices, and discounts
- Get automatic net/VAT/total calculations
- Preview the PDF before download
- Reuse buyer details with autofill
- Customize branding with logo and accent color
- Keep everything local in your browser storage

No external upload is required for offer creation. Your data remains on your device.

## Detailed Description (HR)

Izradite uredne poslovne ponude izravno u pregledniku i izvezite ih u PDF u nekoliko sekundi.

`Izrada ponuda` ubrzava izradu ponuda kroz jasan workflow:

- Upravljanje vise izdavatelja
- Uvoz podataka izdavatelja iz e-racun XML datoteke
- Stavke ponude s kolicinom, jed. cijenom i popustom
- Automatski izracun neta, PDV-a i ukupnog iznosa
- Pregled PDF-a prije preuzimanja
- Autofill kupaca iz prethodnih ponuda
- Brendiranje kroz logo i boju
- Lokalna pohrana podataka u pregledniku

Za izradu ponude nije potreban upload na vanjske servise. Podaci ostaju na vasem uredjaju.

## Submission Text (Reviewer Notes, EN)

Single purpose:
This extension is used to create and export business offer PDFs from user-provided data.

Permission usage:
- `storage`: saves user settings, issuer profiles, buyer autofill entries, and generated offer metadata locally.
- `sidePanel`: enables running the extension UI in Chrome side panel mode.
- `activeTab`: obtains the current tab context so the side panel can open on the active tab.

Data handling:
- No external API calls are required for offer creation.
- No analytics, ads, or tracking scripts.
- No host permissions.
- No content scripts.

Osnovna namjena:
Prosirenje sluzi za izradu i izvoz poslovnih PDF ponuda iz podataka koje unosi korisnik.

Koristenje dozvola:

Obrada podataka:
